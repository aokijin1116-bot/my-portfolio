#### 学んだコマンド
| コマンド | 意味 |
|---|---|
| `mkdir` | フォルダを作る |
| `cd` | フォルダに移動する |
| `ls` | ファイル一覧を表示する |
| `touch` | ファイルを作る |
| `open` | ファイルを開く |
| `git init` | Gitの管理を開始する |
| `git add` | セーブ対象に追加する |
| `git commit` | セーブを確定する |
| `git push` | GitHubにアップロードする |
